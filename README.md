## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/LearnTypeScript.git
```

### Navigate to the Project

```bash
cd LearnTypeScript
```

### Install Dependencies

```bash
npm install
```

### Compile TypeScript

```bash
npx tsc
```

### Run JavaScript Output

```bash
node dist/index.js
```
